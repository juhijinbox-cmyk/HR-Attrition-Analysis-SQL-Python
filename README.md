# HR Attrition Analysis using SQL & Python

## Project Overview
This project analyzes employee attrition using SQL and Python to identify the key factors that influence employee turnover. The analysis answers important business questions and provides actionable business insights based on the data.

## Objectives
- Identify the department with the highest attrition rate.
- Analyze average monthly income by job role.
- Study the impact of overtime on employee attrition.
- Find which age group has the highest attrition.
- Compare attrition rates by gender.
- Identify job roles with the highest attrition count.
- Analyze whether distance from home affects employee attrition.

## Tools & Technologies
- Python
- SQL
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Business Insights
- Sales department has the highest attrition rate.
- Employees working overtime are significantly more likely to leave.
- Employees under 25 have the highest attrition rate.
- Sales Representatives have the highest attrition rate among job roles.
- Employees living farther from the office show higher attrition.
